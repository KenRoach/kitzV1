### Uber Direct API
- **Priority:** Medium
- **ID:** PKB-182
- **URL:** https://developer.uber.com/
- **Why KITZ needs it:** On-demand delivery API for businesses

**Extracted Intelligence:**

Uber Direct provides API for integrating on-demand delivery. Create, manage, monitor deliveries programmatically. Real-time webhooks: delivery_status (on change), courier_update (every 20s with location). Sandbox for testing. Production requires billing + account approval. Designed for 1,000+ deliveries/month. Documentation at developer.uber.com/docs/deliveries/.

**KITZ Application:** Enable on-demand delivery for KITZ users. Real-time tracking via webhooks. Volume requirement may limit very small businesses.
