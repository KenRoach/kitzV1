# IBM — Tech Provider Knowledge Base

## Company Overview
- **Founded**: 1911 (Armonk, NY)
- **Ticker**: IBM (NYSE)
- **Revenue**: ~$62B (FY2024)
- **CEO**: Arvind Krishna
- **Employees**: ~280,000
- **Key spin-off**: Kyndryl (2021) — managed infrastructure services
- **Key acquisition**: HashiCorp ($6.4B, 2024) — infrastructure automation (Terraform, Vault, Consul)

## Product Portfolio

### Software
- **Red Hat** (acquired 2019, $34B):
  - RHEL (Red Hat Enterprise Linux): Enterprise Linux standard
  - OpenShift: Enterprise Kubernetes platform (#1 enterprise K8s)
  - Ansible: IT automation (playbooks, Tower/Controller)
  - Quay: Container registry
  - Advanced Cluster Management: Multi-cluster K8s management
- **IBM watsonx**: AI platform
  - watsonx.ai: Foundation model studio (Granite models, open-source models)
  - watsonx.data: Lakehouse (Presto + Iceberg)
  - watsonx.governance: AI lifecycle governance, bias detection, compliance
- **IBM Cloud Pak**: Containerized middleware on OpenShift
  - Cloud Pak for Data (analytics/AI)
  - Cloud Pak for Integration (API, messaging, event)
  - Cloud Pak for Security (SIEM, SOAR, QRadar)
  - Cloud Pak for Business Automation (workflow, decisions, content)
- **IBM MQ**: Enterprise messaging middleware
- **IBM Db2**: Enterprise relational database
- **IBM CICS/IMS**: Mainframe transaction processing
- **IBM WebSphere**: Application server (being replaced by Open Liberty)
- **Apptio (acquired 2023)**: IT financial management, FinOps

### Infrastructure
- **IBM Power**: POWER10 servers (AIX, IBM i, Linux). Mission-critical workloads
- **IBM Z (Mainframe)**: z16/z17 — 70%+ of Fortune 500 run on Z. AI on-chip (Telum processor)
- **IBM Storage**: FlashSystem (all-flash arrays), Fusion HCI, tape (TS series), SDS (Spectrum)
- **IBM Cloud**: Public cloud (smaller than AWS/Azure/GCP but strong in regulated industries)

### Consulting
- **IBM Consulting**: ~$20B revenue. Strategy, technology, managed operations
- **Focus areas**: Cloud migration, AI implementation, cybersecurity, SAP/Salesforce implementation
- **IBM Garage**: Co-creation methodology (design thinking + agile + DevOps)

### Security
- **QRadar SIEM**: Security information & event management
- **QRadar SOAR**: Security orchestration, automation, response
- **Guardium**: Data security & compliance (database activity monitoring)
- **Verify**: Identity & access management (IAM)
- **MaaS360**: Unified endpoint management (MDM)
- **X-Force**: Threat intelligence + incident response
- **IBM Security Randori**: Attack surface management

## SMB Relevance
- **IBM is primarily enterprise-focused** — less direct SMB play than Dell/HP/Microsoft
- **Red Hat**: SMBs use RHEL, Ansible, OpenShift (especially developer-oriented companies)
- **IBM Cloud**: Lite/free tier available, but less SMB adoption vs AWS/Azure/GCP
- **watsonx**: Granite models available on IBM Cloud (competitive AI pricing)
- **IBM LinuxONE**: Linux-on-Z — consolidation play for mid-market
- **HashiCorp products**: Terraform, Vault widely used by SMB dev teams (free tiers available)
- **IBM SkillsBuild**: Free tech education platform (AI, cybersecurity, cloud)

## Key Differentiators
- **Enterprise trust**: 100+ years, trusted by banks, governments, Fortune 500
- **Mainframe monopoly**: IBM Z runs 70%+ of global transaction processing
- **Red Hat ecosystem**: RHEL, OpenShift, Ansible — dominant in enterprise Linux/K8s/automation
- **HashiCorp acquisition**: Terraform + Vault = infrastructure-as-code + secrets management standard
