# HP Inc. & Hewlett Packard Enterprise — OEM & Tech Provider Knowledge Base

## Company Overview (Two Companies Since 2015 Split)

### HP Inc. (HPQ)
- **Focus**: PCs, printers, peripherals
- **Revenue**: ~$54B (FY2024)
- **HQ**: Palo Alto, CA
- **CEO**: Enrique Lores
- **Employees**: ~58,000

### Hewlett Packard Enterprise (HPE)
- **Focus**: Servers, networking, storage, hybrid cloud
- **Revenue**: ~$30B (FY2024, including Juniper acquisition)
- **HQ**: Spring, TX
- **CEO**: Antonio Neri
- **Employees**: ~62,000
- **Key acquisition**: Juniper Networks ($14B, 2024) — enterprise networking + AI-native

## HP Inc. — Product Portfolio

### Business PCs
- **EliteBook**: Premium business laptops (600/800/1000 series). HP Sure Sense AI, Sure View privacy screen
- **ProBook**: Mid-range business laptops (400/600 series)
- **EliteDesk / ProDesk**: Business desktops (Mini, SFF, Tower, AIO)
- **Elite Dragonfly**: Ultra-premium 2-in-1 business laptop (~1kg)
- **ZBook**: Mobile workstations (Firefly, Power, Fury, Studio). ISV-certified
- **Z Desktop Workstations**: Z2, Z4, Z6, Z8 towers. Dual GPU support

### Printers & Managed Print Services
- **LaserJet Enterprise / Pro**: Business laser printers & MFPs
- **OfficeJet Pro**: Business inkjet (HP+ smart printing)
- **HP Managed Print Services (MPS)**: Fleet management, toner auto-replenishment, security
- **HP Instant Ink / All-In Plan**: Subscription printing (ink + printer)
- **HP Smart App**: Mobile printing, scanning, fax

### Peripherals
- **Poly (acquired 2023)**: Video conferencing (Studio, Sightline), headsets (Voyager, Blackwire), phones
- **HP Monitors**: E-series (business), Z-series (creative), M-series (conferencing with Poly)
- **Docking**: Thunderbolt G4/G5 docks, USB-C hubs

## HPE — Product Portfolio

### Compute
- **ProLiant**: Industry-standard rack & tower servers (DL, ML series). Gen11 (Intel SPR / AMD Genoa)
- **Synergy**: Composable infrastructure (blades + fabric)
- **Edgeline**: Ruggedized edge compute
- **ProLiant for AI**: DL380a Gen11 (NVIDIA L40S), ProLiant Compute XD685 (8x GPU)

### Storage
- **HPE Alletra**: Cloud-native, all-flash storage (Alletra 5000/6000/9000/MP)
- **HPE StoreEasy**: Windows-based NAS for file workloads
- **Zerto (acquired)**: Continuous data protection, DR, ransomware recovery

### Networking (+ Juniper)
- **Aruba**: Wireless (APs, controllers), wired switches (CX series), SD-WAN, SASE
- **Aruba Central**: Cloud-managed networking, AIOps
- **Juniper (acquired 2024)**: EX switches, SRX firewalls, Mist AI, Apstra intent-based networking

### HPE GreenLake
- **HPE's as-a-service platform** — consumption-based IT (compute, storage, networking, ML Ops)
- **GreenLake Central**: Single dashboard for hybrid cloud management
- **GreenLake for Private Cloud**: VMware + HPE infrastructure
- **GreenLake for Data**: Unified analytics (Ezmeral discontinued, pivoted to partner ecosystem)
- **Pay-per-use**: Metered billing, buffer capacity pre-deployed on-prem

## SMB Programs & Relevance

### HP Inc.
- **HP Amplify Partner Program**: Synergy, Power, Select tiers
- **HP for Small Business**: hp.com/sbso — curated bundles, financing (HP Financial Services)
- **HP Subscription**: Device-as-a-Service (DaaS) for PCs — monthly per-device fee, includes support + lifecycle
- **HP Wolf Security**: Hardware-enforced endpoint security (Sure Click, Sure Start BIOS protection)
- **HP Anyware**: Remote workstation access (formerly Teradici)

### HPE
- **HPE Partner Ready**: Partner program (Business, Volume, Service)
- **HPE SMB Solutions**: Entry servers (ProLiant ML30/ML110), Aruba Instant On (SMB networking)
- **Aruba Instant On**: Affordable managed Wi-Fi/switches for SMBs (cloud-managed, app-controlled)
- **HPE Small Business Solutions**: hpe.com/smb — bundles, financing, quick-start guides
- **HPE Financial Services**: Leasing, consumption models, IT asset disposition

## APIs & Integration Points

### HP Inc.
- **HP Device Manager API**: Fleet management for thin clients
- **HP Smart Device Services API**: Printer fleet telemetry, alerts, supply levels
- **HP MPS REST API**: Managed print services integration
- **HP Wolf Security Controller API**: Threat containment policy management
- **HP Workforce Experience Platform API**: Employee device analytics

### HPE
- **iLO (Integrated Lights-Out) RESTful API**: Server management (Redfish-compliant)
- **HPE OneView API**: Infrastructure automation, templates, profiles
- **GreenLake API**: Programmatic provisioning, metering, subscription management
- **Aruba Central API**: Network config, monitoring, troubleshooting
- **InfoSight API**: AI-driven predictive analytics for storage & servers

## Security & Compliance
- **HP Wolf Security**: Hardware root of trust, Sure Start (BIOS self-healing), Sure Click (micro-VM browser isolation), Sure Sense (deep learning malware prevention)
- **HPE Silicon Root of Trust**: Firmware integrity from silicon up
- **Aruba ClearPass**: Network access control (NAC), zero-trust segmentation
- **Compliance**: FIPS 140-2/3, Common Criteria, NIST, SOC 2 (GreenLake), TAA-compliant
