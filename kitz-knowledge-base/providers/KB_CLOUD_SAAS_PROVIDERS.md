## Shopify

### Overview
- **Ticker**: SHOP (NYSE/TSX) | **Revenue**: ~$8B | **HQ**: Ottawa, Canada
- **CEO**: Tobi Lutke (founder) | **Employees**: ~12,000
- **Market position**: #1 DTC e-commerce platform, #2 US e-commerce (behind Amazon)

### Key Products
- **Shopify Basic**: $39/mo — online store, unlimited products, 2 staff accounts
- **Shopify Standard**: $105/mo — professional reports, 5 staff accounts
- **Shopify Advanced**: $399/mo — advanced analytics, custom reports, 15 staff accounts
- **Shopify Plus**: From $2,000/mo — enterprise (Checkout extensibility, B2B, multi-store)
- **Shopify POS**: Point-of-sale hardware + software (Lite free with plan, Pro $89/location/mo)
- **Shop Pay**: Accelerated checkout (4x higher conversion)
- **Shopify Payments**: Built-in payment processing (2.9% + $0.30 Basic)
- **Shopify Fulfillment Network**: Warehousing + shipping (US)
- **Shopify Capital**: Business loans/cash advances for merchants
- **Shopify Markets**: International selling (multi-currency, duties, taxes)
- **Shopify Flow**: No-code automation (free with Standard+)

### APIs
- **Admin API (REST + GraphQL)**: Full store management (products, orders, customers, inventory)
- **Storefront API (GraphQL)**: Custom storefronts (headless commerce)
- **Hydrogen + Oxygen**: React-based headless storefront framework + hosting
- **Shopify App Bridge**: Embedded app SDK
- **Shopify Webhooks**: Real-time event notifications

### LatAm: Available in Spanish/Portuguese. Shopify Payments in Mexico, Brazil (coming). Strong adoption in LatAm DTC brands

## Stripe

### Overview
- **Private company** | **Valuation**: ~$70B+ | **Revenue**: ~$18B+ | **HQ**: San Francisco + Dublin
- **CEO**: Patrick Collison (co-founder) | **Employees**: ~8,000
- **Market position**: #1 developer-first payments platform

### Key Products
- **Payments**: Card processing, wallets (Apple Pay, Google Pay), local payment methods
- **Stripe Connect**: Marketplace/platform payments (onboard sellers, split payments)
- **Stripe Billing**: Subscriptions, invoicing, usage-based billing, revenue recognition
- **Stripe Tax**: Automated tax calculation (sales tax, VAT, GST) in 50+ countries
- **Stripe Atlas**: Incorporate a US company online ($500 one-time)
- **Stripe Treasury**: Banking-as-a-Service (embedded financial accounts)
- **Stripe Issuing**: Create virtual/physical cards (expense management, payouts)
- **Stripe Radar**: ML-based fraud detection (included with Payments)
- **Stripe Identity**: ID verification (document + selfie)
- **Stripe Financial Connections**: Bank account linking (for ACH, verification)
- **Stripe Climate**: Carbon removal with every transaction

### Pricing
- **Standard**: 2.9% + $0.30 per successful charge (US cards)
- **International**: +1.5% for cross-border, +1% for currency conversion
- **Connect**: $2/active connected account/mo (Standard), custom pricing for Custom
- **Invoicing**: 0.4% per paid invoice (first 25 free/mo)
- **Billing**: 0.5% on recurring charges (Starter, included up to $1M)

### APIs
- **Stripe API**: REST API for everything (payments, subscriptions, customers, products, etc.)
- **Stripe.js / Elements**: Pre-built UI components for checkout
- **Stripe SDKs**: Node.js, Python, Ruby, PHP, Java, Go, .NET
- **Stripe Webhooks**: Real-time event notifications (500+ event types)
- **Stripe Apps**: Build apps that run inside Stripe Dashboard

### LatAm: Available in Mexico, Brazil (Stripe Atlas available globally). Supports OXXO, boleto, PIX payment methods

## Twilio

### Overview
- **Ticker**: TWLO (NYSE) | **Revenue**: ~$4.2B | **HQ**: San Francisco, CA
- **CEO**: Khozema Shipchandler | **Employees**: ~5,500
- **Market position**: #1 cloud communications API platform (CPaaS)

### Key Products
- **Programmable Messaging**: SMS, MMS, WhatsApp, Facebook Messenger
- **Programmable Voice**: Make/receive calls, IVR, SIP trunking
- **SendGrid**: Email delivery (transactional + marketing)
- **Twilio Verify**: OTP/2FA verification (SMS, email, push, TOTP)
- **Twilio Conversations**: Multi-channel messaging SDK
- **Twilio Flex**: Programmable contact center
- **Twilio Segment**: Customer Data Platform (CDP) — acquired 2020
- **TaskRouter**: Intelligent task routing for contact centers

### Pricing
- **SMS (US)**: $0.0079/outbound, $0.0075/inbound
- **Voice (US)**: $0.014/min outbound, $0.0085/min inbound
- **WhatsApp**: $0.005/message (user-initiated), $0.05/message (business-initiated)
- **SendGrid**: Free (100 emails/day), Essentials ($19.95/mo for 50K)
- **Phone numbers**: $1.15/mo (US local), $2/mo (US toll-free)
- **Verify**: $0.05/verification

### APIs
- **REST API**: Full programmable access to all Twilio services
- **SDKs**: Node.js, Python, Ruby, PHP, Java, C#, Go
- **TwiML**: XML markup for voice/messaging call flows
- **Twilio Functions**: Serverless compute for Twilio logic
- **Studio**: Visual flow builder (drag-and-drop IVR, chatbots)

### LatAm: Local numbers in 15+ LatAm countries. WhatsApp Business API widely used. Offices in Bogota, Mexico City

## Intuit

### Overview
- **Ticker**: INTU (NASDAQ) | **Revenue**: ~$16B | **HQ**: Mountain View, CA
- **CEO**: Sasan Goodarzi | **Employees**: ~27,000
- **Market position**: #1 SMB accounting (QuickBooks), #1 consumer tax (TurboTax)

### Key Products
- **QuickBooks Online**: Cloud accounting for SMBs
  - Simple Start ($30/mo), Essentials ($60/mo), Plus ($90/mo), Advanced ($200/mo)
  - Invoicing, expense tracking, bank feeds, payroll, inventory, reports
- **QuickBooks Payments**: Accept card payments (2.4% + $0.25 invoiced, 2.9% + $0.25 keyed)
- **QuickBooks Payroll**: Payroll processing (from $45/mo + $6/employee)
- **QuickBooks Commerce**: Inventory + order management (multichannel)
- **Mailchimp** (acquired 2021): Email marketing, automation, landing pages
  - Free (500 contacts), Essentials ($13/mo), Standard ($20/mo), Premium ($350/mo)
- **TurboTax**: Consumer/self-employed tax filing
- **Credit Karma**: Personal finance (free credit scores, loans, insurance)
- **Intuit Assist**: AI assistant across all Intuit products

### APIs
- **QuickBooks API**: REST API for accounting data (invoices, customers, items, payments, reports)
- **Mailchimp API**: REST API for email campaigns, automations, audiences
- **QuickBooks Payments API**: Charge cards, bank transfers within QB

### LatAm: QuickBooks available in Mexico, Brazil (QuickBooks Online Global). Mailchimp available globally. Limited localization vs competitors like Alegra, Siigo, Contabilizei
