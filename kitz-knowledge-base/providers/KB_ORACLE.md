# Oracle — Tech Provider Knowledge Base

## Company Overview
- **Founded**: 1977 by Larry Ellison, Bob Miner, Ed Oates (Austin, TX — moved from Redwood City 2021)
- **Ticker**: ORCL (NYSE)
- **Revenue**: ~$56B (FY2025)
- **CEO**: Safra Catz; Larry Ellison = Chairman/CTO
- **Employees**: ~164,000
- **Market position**: #1 enterprise database, #3/#4 cloud (fastest-growing)

## Product Portfolio

### Database
- **Oracle Database 23ai**: Flagship RDBMS. RAC (Real Application Clusters), Multitenant, In-Memory, AI Vectors
- **Autonomous Database**: Self-driving, self-securing, self-repairing (OLTP + DW + JSON)
- **MySQL**: World's most popular open-source DB (owned by Oracle). HeatWave for analytics + ML
- **Oracle NoSQL**: Distributed key-value/document/column database
- **Oracle Database Free**: Free edition (up to 12GB user data)

### Oracle Cloud Infrastructure (OCI)
- **Compute**: Flexible VMs (Intel, AMD, Arm/Ampere A1), bare metal, GPU (NVIDIA A100/H100)
- **Autonomous DB**: Managed Oracle Database in cloud (serverless option)
- **Oracle Container Engine (OKE)**: Managed Kubernetes
- **Oracle Functions**: Serverless (Fn-based)
- **Networking**: VCN, Load Balancer, FastConnect (dedicated connectivity)
- **Storage**: Block, Object, File, Archive
- **OCI AI Services**: Language, Vision, Speech, Document Understanding, Anomaly Detection
- **OCI Generative AI**: Cohere, Meta Llama models hosted on OCI
- **Always Free Tier**: 2 VMs (Arm A1, 4 OCPU/24GB RAM total), 200GB storage, Autonomous DB, more

### Applications (SaaS)
- **Oracle Fusion Cloud ERP**: Financials, procurement, project management, risk management
- **Oracle Fusion Cloud HCM**: HR, payroll, talent management, workforce management
- **Oracle Fusion Cloud SCM**: Supply chain planning, manufacturing, logistics, IoT
- **Oracle NetSuite**: Cloud ERP for SMBs/mid-market (#1 cloud ERP). Financials, CRM, e-commerce, inventory
- **Oracle Fusion Cloud CX**: Sales, Service, Marketing, Commerce (CRM)
- **Oracle Health (Cerner)**: Healthcare EHR (acquired 2022, $28B)

### Middleware & Development
- **WebLogic**: Java application server
- **GraalVM**: Polyglot VM (Java, JavaScript, Python, Ruby, LLVM)
- **Oracle APEX**: Low-code app development platform (free with Oracle DB)
- **Oracle Integration Cloud**: iPaaS
- **Oracle SOA Suite**: Enterprise service bus, BPEL orchestration

## SMB Programs & Relevance

  - SuiteSuccess: Industry-specific pre-configured editions (services, retail, manufacturing, wholesale, etc.)

## Key Differentiators
- **Database dominance**: Oracle DB runs mission-critical systems at most Fortune 500 companies
- **OCI price-performance**: Consistently cheaper than AWS/Azure for compute + storage + egress
- **Always Free tier**: Most generous perpetual free tier of any cloud (real VMs, real database)
- **NetSuite**: #1 cloud ERP for SMB/mid-market, 40,000+ customers
- **Autonomous Database**: Self-tuning, self-patching, self-securing — reduces DBA overhead
- **Multi-cloud strategy**: Oracle DB on Azure (OracleDB@Azure), AWS, GCP
- **AI infrastructure**: Major NVIDIA GPU cloud partner (Elon Musk's xAI, TikTok train on OCI)

## Security & Compliance
- **Oracle Cloud Guard**: Automated threat detection and remediation
- **Oracle Data Safe**: Database security assessment, auditing, data masking
- **Maximum Security Zones**: Enforced security policies (cannot be overridden)
- **Dedicated Region**: OCI region deployed in customer's data center
- **Compliance**: SOC 1/2/3, ISO 27001, PCI-DSS, HIPAA, FedRAMP (IL5), GDPR, LGPD
- **Database Vault**: Privileged user access control (DBA can't see data)
- **Transparent Data Encryption**: Encryption at rest for all data (default)
