# Salesforce — Tech Provider Knowledge Base

## Company Overview
- **Founded**: 1999 by Marc Benioff (San Francisco, CA)
- **Ticker**: CRM (NYSE)
- **Revenue**: ~$35B (FY2025)
- **CEO**: Marc Benioff
- **Employees**: ~72,000
- **Market position**: #1 CRM platform globally (~23% market share). Pioneered SaaS model

## Product Portfolio

### Sales Cloud
- **Core CRM**: Leads, contacts, accounts, opportunities, pipeline management
- **Einstein AI**: Lead scoring, opportunity insights, activity capture, email insights
- **Revenue Intelligence**: Forecasting, pipeline inspection, deal insights
- **Sales Engagement**: Cadences, email sequences, dialer
- **CPQ (Configure, Price, Quote)**: Complex quoting, approvals, contract management
- **Pricing**: Starter ($25/user/mo), Professional ($80), Enterprise ($165), Unlimited ($330)

### Service Cloud
- **Case management**: Omnichannel routing, entitlements, milestones
- **Knowledge Base**: Article management, self-service portal
- **Field Service**: Scheduling, dispatch, mobile workforce
- **Einstein Bots**: AI chatbots for customer service
- **Pricing**: Starter ($25), Professional ($80), Enterprise ($165), Unlimited ($330)

### Marketing Cloud
- **Account Engagement (Pardot)**: B2B marketing automation
- **Marketing Cloud Engagement**: B2C email, SMS, push, journey builder
- **Data Cloud**: CDP (Customer Data Platform) — unified customer profiles
- **Advertising Studio**: Audience targeting across Google, Facebook, LinkedIn
- **Pricing**: Varies significantly (~$1,250-$65,000/mo depending on contacts/features)

### Commerce Cloud
- **B2C Commerce**: Storefront, catalog, checkout, order management
- **B2B Commerce**: Self-service portals, account-based pricing, reorder
- **Order Management**: Distributed order management, fulfillment
- **Pricing**: Revenue share model (1-3% of GMV)

### Platform & Developer
- **Salesforce Platform**: Low-code/no-code app builder (Lightning App Builder, Flow)
- **Apex**: Proprietary programming language (Java-like)
- **Lightning Web Components (LWC)**: Modern web component framework
- **Heroku**: PaaS for custom apps (Node.js, Python, Ruby, Java, Go)
- **MuleSoft**: API management + iPaaS (integration platform)
- **Tableau**: Business intelligence & data visualization
- **Slack**: Team messaging & collaboration (acquired 2021, $27.7B)

### Agentforce (AI)
- **Agentforce**: Autonomous AI agents for sales, service, marketing, commerce
- **Einstein GPT**: Generative AI across all Salesforce clouds
- **Einstein Copilot**: Conversational AI assistant within Salesforce
- **Data Cloud**: Real-time customer data unification + AI grounding
- **Prompt Builder**: Custom AI prompts using CRM data
- **Einstein Trust Layer**: Secure AI with data masking, toxicity detection, audit trail

## SMB Programs & Relevance
- **Salesforce Starter Suite**: $25/user/mo — unified sales, service, marketing CRM for small teams
- **Salesforce Pro Suite**: $100/user/mo — more customization, automation, quoting
- **Salesforce Essentials** (discontinued, replaced by Starter): Was the entry-level SMB product
- **AppExchange**: 7,000+ apps (many free) — extend Salesforce without code
- **Trailhead**: Free gamified learning platform (admin, developer, marketing certifications)
- **Salesforce for Startups**: Discounted licenses (up to 80% off for qualifying startups)
- **Small Business Hub**: salesforce.com/small-business

## Partner Program (Salesforce Partner Program)
- **Tracks**: Consulting, ISV (AppExchange), Reseller
- **Tiers**: Base, Ridge, Crest, Summit
- **Benefits**: Leads, MDF, co-marketing, training, demo orgs
- **AppExchange**: Marketplace for ISV apps (free listing, 15-25% revenue share)
- **Consulting ecosystem**: Massive — Accenture, Deloitte, Cognizant, Capgemini, PwC, hundreds of boutiques

## APIs & Integration Points
- **Salesforce REST API**: CRUD for all standard/custom objects
- **Salesforce SOAP API**: Legacy enterprise integration
- **Bulk API 2.0**: High-volume data operations (millions of records)
- **Streaming API / Platform Events**: Real-time data change notifications
- **Connect API**: Chatter, communities, files, topics
- **Metadata API**: Deploy/retrieve customizations
- **Tooling API**: Apex code, Visualforce pages, Lightning components
- **MuleSoft Anypoint**: 400+ pre-built connectors (SAP, Workday, Netsuite, etc.)
- **Salesforce Functions**: Custom serverless compute (Apex/Node.js/Java)
- **Einstein API**: Prediction, sentiment, intent, object detection
- **Data Cloud API**: Ingest, query, segment unified customer data

## Security & Compliance
- **Salesforce Shield**: Platform encryption, event monitoring, field audit trail ($additional)
- **Einstein Trust Layer**: Data masking, prompt defense, toxicity detection for AI
- **Hyperforce**: Public cloud deployment (AWS/Azure) with data residency controls
- **MFA enforcement**: Required for all Salesforce logins (since 2022)
- **Compliance**: SOC 1/2, ISO 27001/27017/27018, PCI-DSS, HIPAA, FedRAMP, GDPR, LGPD (Brazil)
- **Health Check**: Built-in security assessment tool
- **Named credentials**: Secure external API callout management
