## Fortinet

### Overview
- **Ticker**: FTNT (NASDAQ) | **Revenue**: ~$6B | **HQ**: Sunnyvale, CA
- **CEO**: Ken Xie (founder) | **Employees**: ~14,000
- **Market position**: #1 network firewall (units shipped), #1 SD-WAN

### Key Products
- **FortiGate NGFW**: Next-gen firewalls (40/60/80/100/200/600/1800/3000/4000 series)
- **FortiSwitch**: Managed switches (integrated with FortiGate)
- **FortiAP**: Managed Wi-Fi access points
- **FortiManager / FortiAnalyzer**: Centralized management + analytics
- **FortiSASE**: Cloud-delivered SASE (ZTNA, SWG, CASB, SD-WAN)
- **FortiEDR / FortiXDR**: Endpoint detection & response
- **FortiMail**: Email security gateway
- **FortiWeb**: Web application firewall (WAF)
- **Security Fabric**: Unified security architecture across all FortiProducts

### SMB Relevance
- **FortiGate 40F/60F/80F**: Purpose-built for SMBs ($300-$1,200 + FortiGuard license)
- **Unified Threat Management (UTM)**: All-in-one box (firewall, VPN, AV, IPS, web filter, SD-WAN)
- **FortiGate Cloud**: Cloud-managed for MSPs managing SMB clients
- **Pricing**: FortiGate 60F ~$500 + FortiGuard Bundle ~$350/yr

### LatAm: Offices in Mexico, Brazil, Colombia, Argentina. Strong channel/MSP ecosystem

## Palo Alto Networks

### Overview
- **Ticker**: PANW (NASDAQ) | **Revenue**: ~$8B | **HQ**: Santa Clara, CA
- **CEO**: Nikesh Arora | **Employees**: ~16,000
- **Market position**: #1 cybersecurity company by market cap, premium NGFW

### Key Products
- **Strata (Network Security)**: PA-series NGFW (PA-400/800/1400/3400/5400/7500)
- **Prisma Cloud**: Cloud-native security (CNAPP — CSPM, CWPP, CIEM, IaC scanning)
- **Prisma Access**: Cloud-delivered SASE (ZTNA, SWG, CASB)
- **Prisma SD-WAN**: Software-defined WAN (acquired CloudGenix)
- **Cortex XDR**: Extended detection & response
- **Cortex XSOAR**: Security orchestration, automation, response
- **Cortex XSIAM**: AI-driven SOC platform
- **Unit 42**: Threat intelligence + incident response consulting
- **Wildfire**: Cloud-based malware analysis sandbox

### SMB Relevance
- **PA-410/PA-415/PA-440**: Branch/SMB firewalls ($1,000-$3,000 + subscription)
- **Prisma Access for small teams**: Cloud-managed SASE starting ~$75/user/yr
- **Premium positioning**: More expensive than Fortinet, but top-tier security
- **NextWave Partner Program**: MSPs can manage Palo Alto for SMB clients

### LatAm: Offices in Sao Paulo, Mexico City, Bogota. Growing LatAm channel

## Cloudflare

### Overview
- **Ticker**: NET (NYSE) | **Revenue**: ~$1.7B | **HQ**: San Francisco, CA
- **CEO**: Matthew Prince (co-founder) | **Employees**: ~4,000
- **Market position**: Largest global edge network (330+ cities), #1 CDN for websites

### Key Products
- **CDN**: Content delivery, caching, DDoS protection (free tier available)
- **DNS**: Fastest public DNS (1.1.1.1), authoritative DNS
- **WAF**: Web application firewall
- **Zero Trust / Cloudflare One**: ZTNA, SWG, CASB, email security, browser isolation
- **Workers**: Serverless edge compute (V8 isolates, KV storage, D1 database, R2 object storage)
- **Pages**: Static site hosting (free tier)
- **Stream**: Video streaming + encoding
- **Images**: Image optimization CDN
- **AI Gateway**: AI request caching, rate limiting, logging
- **Workers AI**: Inference on Cloudflare's edge (Llama, Mistral, Stable Diffusion)

### SMB Relevance
- **Free tier**: DNS, CDN, DDoS protection, 100K Workers requests/day — extremely generous
- **Pro plan**: $20/mo (WAF, image optimization, analytics)
- **Business plan**: $200/mo
- **Registrar**: At-cost domain registration (no markup)
- **Workers/Pages**: Free hosting for small apps — ideal for SMBs

### LatAm: Edge nodes in 20+ LatAm cities. Widely used by LatAm startups/SMBs

## Ubiquiti

### Overview
- **Ticker**: UI (NYSE) | **Revenue**: ~$2B | **HQ**: New York, NY
- **CEO**: Robert Pera (founder) | **Employees**: ~1,500
- **Market position**: #1 prosumer/SMB networking (massive community, direct-to-customer model)

### Key Products
- **UniFi**: Networking ecosystem
  - Dream Machine (UDM): Router/firewall/switch/AP all-in-one ($130-$500)
  - UniFi Switches: Managed PoE switches (8/16/24/48 port)
  - UniFi APs: Wi-Fi 6/6E/7 access points ($99-$399)
  - UniFi Cameras (Protect): PoE security cameras + NVR
  - UniFi Network Application: Free management controller (self-hosted or Cloud)
- **UniFi Talk**: VoIP phone system (PBX built into UniFi)
- **UniFi Access**: Door access control (NFC readers, hub)
- **UISP**: ISP/MSP network management (airMAX, airFiber, EdgeRouter/EdgeSwitch)

### SMB Relevance
- **Price disruptor**: Enterprise-grade features at consumer prices
- **No licensing fees**: Buy hardware once, no recurring subscription
- **Self-managed**: UniFi controller is free (cloud or local)
- **Huge SMB adoption**: Restaurants, offices, retail stores, Airbnbs worldwide
- **LatAm**: Very popular for SMB deployments due to price-performance
