# Cisco Systems — OEM & Tech Provider Knowledge Base

## Company Overview
- **Founded**: 1984 (San Jose, CA)
- **Ticker**: CSCO (NASDAQ)
- **Revenue**: ~$54B (FY2024)
- **CEO**: Chuck Robbins
- **Employees**: ~85,000
- **Market position**: #1 enterprise networking, #1 network security, major collaboration vendor
- **Key acquisition**: Splunk ($28B, 2024) — observability + security analytics

## Product Portfolio

### Networking
- **Catalyst switches**: Campus/branch switching (Catalyst 9200/9300/9400/9500/9600). Wi-Fi 6E/7 APs (Catalyst 9100 series)
- **Nexus switches**: Data center switching (9000/3000 series). ACI fabric
- **Meraki**: Cloud-managed networking (switches, APs, cameras, SD-WAN). Dashboard-driven, ideal for SMBs
- **ISR/Catalyst 8000 routers**: Branch/WAN routers, SD-WAN capable
- **ThousandEyes**: Internet/cloud intelligence, network path visualization

### Security
- **Cisco Secure Firewall (Firepower)**: NGFW (1000/2100/3100/4200/9300 series)
- **Umbrella**: Cloud-delivered DNS security, SIG, CASB
- **Secure Endpoint (AMP)**: EDR/XDR endpoint protection
- **Duo**: Multi-factor authentication, zero-trust access
- **Secure Access (SASE)**: SSE + SD-WAN converged
- **Identity Services Engine (ISE)**: Network access control (NAC), segmentation
- **Talos**: Threat intelligence unit (one of the world's largest)
- **Splunk (acquired 2024)**: SIEM, SOAR, observability, log analytics

### Collaboration
- **Webex Suite**: Video conferencing, messaging, calling, contact center, events, whiteboarding
- **Webex Calling**: Cloud PBX (replaces on-prem Cisco UCM)
- **Webex Contact Center**: CCaaS with AI agent assist
- **Room devices**: Webex Board, Room Kit, Desk Hub/Pro, Room Bar
- **Cisco Phones**: IP phones (8800/9800 series, video phones)

### Compute (UCS)
- **UCS C-Series**: Rack servers (Cisco-branded, Intel/AMD)
- **UCS X-Series**: Modular computing (X9508 chassis)
- **HyperFlex**: HCI (Cisco-proprietary, being wound down in favor of partnerships)
- **Intersight**: Cloud-based infrastructure management platform (SaaS)

### Observability & Analytics
- **Splunk**: Log analytics, SIEM, IT observability (full-stack)
- **AppDynamics**: Application performance monitoring (APM)
- **ThousandEyes**: Network/internet performance monitoring
- **Intersight Workload Optimizer**: Cross-domain resource optimization

## Key Differentiators
- **Networking dominance**: 50%+ enterprise networking market share
