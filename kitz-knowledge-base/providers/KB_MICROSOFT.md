# Microsoft — Tech Provider Knowledge Base

## Company Overview
- **Founded**: 1975 by Bill Gates & Paul Allen (Redmond, WA)
- **Ticker**: MSFT (NASDAQ)
- **Revenue**: ~$245B (FY2024)
- **CEO**: Satya Nadella
- **Employees**: ~228,000
- **Market cap**: ~$3T+ (world's most valuable company, trades #1/#2 with Apple)

## Product Portfolio — Cloud & Software

### Microsoft 365 (formerly Office 365)
- **Apps**: Word, Excel, PowerPoint, Outlook, OneNote, Access (Windows)
- **Collaboration**: Teams (chat, video, calling, channels), SharePoint, OneDrive, Loop
- **Business plans**: Basic ($6/user/mo), Standard ($12.50), Premium ($22)
- **Enterprise plans**: E1, E3, E5 (up to $57/user/mo)
- **Copilot for Microsoft 365**: AI assistant across all Office apps ($30/user/mo add-on)
- **Exchange Online**: Business email, calendar, contacts (included in M365)
- **Microsoft Bookings**: Online appointment scheduling (included in M365 Business)

### Azure (Cloud Platform)
- **Compute**: VMs (D/E/F/N/H series), Azure Kubernetes Service (AKS), Azure Functions (serverless), Container Apps
- **AI/ML**: Azure OpenAI Service (GPT-4, DALL-E, Whisper), Azure Machine Learning, Cognitive Services, Azure AI Studio
- **Data**: Azure SQL, Cosmos DB, Synapse Analytics, Data Factory, Azure Databricks
- **Storage**: Blob, Files, Queue, Table, Data Lake
- **Networking**: Virtual Network, Load Balancer, Front Door, ExpressRoute, Azure Firewall
- **Identity**: Entra ID (formerly Azure AD), Conditional Access, Privileged Identity Management
- **DevOps**: Azure DevOps (repos, pipelines, boards), GitHub (owned by Microsoft)
- **Regions**: 60+ regions globally including Brazil South, Mexico Central

### Dynamics 365
- **Sales**: CRM, pipeline management, AI-driven insights
- **Customer Service**: Case management, omnichannel, Copilot
- **Business Central**: ERP for SMBs (financials, supply chain, project management)
- **Finance & Supply Chain**: Enterprise ERP
- **Marketing (Customer Insights)**: Journey orchestration, segmentation
- **Field Service**: Work order management, IoT integration
- **Pricing**: Business Central Essentials from $70/user/mo, Premium $100/user/mo

### Windows
- **Windows 11 Pro**: BitLocker, RDP, Hyper-V, Windows Information Protection
- **Windows 11 Enterprise**: Advanced threat protection, AppLocker, Credential Guard
- **Windows 365 Cloud PC**: Cloud-hosted Windows desktops ($28-$158/user/mo)
- **Windows Autopilot**: Zero-touch device deployment
- **Intune**: Endpoint management (MDM/MAM) for Windows, macOS, iOS, Android

### Security
- **Microsoft Defender for Endpoint**: EDR/XDR
- **Microsoft Defender for Cloud**: CSPM, CWPP for Azure/AWS/GCP
- **Microsoft Sentinel**: Cloud-native SIEM/SOAR
- **Microsoft Entra**: Identity & access management (SSO, MFA, Conditional Access)
- **Microsoft Purview**: Data governance, compliance, DLP, information protection
- **Security Copilot**: AI-powered security operations ($4/compute hour)

### Surface (Hardware)
- **Surface Pro**: Detachable 2-in-1 tablet/laptop
- **Surface Laptop**: Traditional clamshell
- **Surface Go**: Budget/education tablet
- **Surface Hub**: Interactive whiteboard (50"/85")
- **Surface for Business**: Enterprise-managed SKUs with Autopilot

## SMB Programs & Relevance
- **Microsoft 365 Business**: Designed for <300 users — email, Office apps, Teams, OneDrive, basic security
- **Microsoft for Startups (Founders Hub)**: Up to $150K Azure credits, M365, GitHub Enterprise, mentorship
- **Dynamics 365 Business Central**: Full ERP built for SMBs (accounting, inventory, sales, purchasing)
- **Power Platform**: Power BI (analytics), Power Apps (low-code apps), Power Automate (RPA/workflows), Power Pages (websites)
- **Microsoft AppSource**: Marketplace for business apps (ISV add-ons for M365/Dynamics)
- **Microsoft Learn**: Free training + certifications (Azure, M365, Dynamics, Security, AI)

## Partner Program (Microsoft AI Cloud Partner Program)
- **Tiers**: Solutions Partner (6 designations: Data & AI, Digital & App Innovation, Infrastructure, Business Applications, Modern Work, Security)
- **Benefits**: IUR (internal use rights), MDF, co-sell with Microsoft sellers, Azure credits
- **CSP (Cloud Solution Provider)**: Partners can bill customers directly for Microsoft services
- **Distribution**: Ingram Micro, TD SYNNEX, Arrow, Pax8, Sherweb
- **Marketplace**: Azure Marketplace + AppSource for listing solutions

## APIs & Integration Points
- **Microsoft Graph API**: Unified API for M365 data (mail, calendar, files, users, Teams, etc.)
- **Azure REST APIs**: Full programmatic control of all Azure services
- **Dynamics 365 Web API**: OData-based CRUD for all Dynamics entities
- **Power Automate connectors**: 1000+ pre-built connectors (Salesforce, SAP, Google, etc.)
- **Bot Framework**: Build conversational bots for Teams, web, Slack
- **Azure OpenAI API**: GPT-4, embeddings, DALL-E, Whisper (same API as OpenAI with enterprise controls)
- **SharePoint REST/Graph API**: Sites, lists, documents, permissions
- **Outlook/Exchange API (Graph)**: Send/read email, calendar events, contacts programmatically
- **OneDrive API (Graph)**: File storage, sharing, sync
- **Intune Graph API**: Device management, compliance policies, app deployment

## Key Differentiators
- **Ecosystem lock-in**: M365 + Azure + Dynamics + Power Platform = full business stack
- **Copilot everywhere**: AI assistant embedded across all products (Office, Dynamics, Security, GitHub)
- **Azure OpenAI**: Enterprise-grade access to GPT-4/o models with data residency + compliance
- **Power Platform**: Low-code tools let SMBs build custom apps without developers
- **Identity backbone**: Entra ID is the world's largest identity platform (700M+ users)
- **GitHub**: World's largest developer platform (100M+ developers)

## Security & Compliance
- **Zero Trust architecture**: Entra ID + Conditional Access + Defender + Intune
- **Compliance**: SOC 1/2/3, ISO 27001/27017/27018, FedRAMP, HIPAA, PCI-DSS, GDPR
- **Data residency**: Multi-Geo for M365, Azure sovereign regions
- **Microsoft Secure Score**: Posture assessment across M365 tenant
- **Customer Lockbox**: Customer approval required before Microsoft can access data
