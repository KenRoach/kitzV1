# Dell Technologies — OEM & Tech Provider Knowledge Base

## Company Overview
- **Founded**: 1984 by Michael Dell (Round Rock, TX)
- **Ticker**: DELL (NYSE), rejoined public markets 2018
- **Revenue**: ~$102B (FY2025), world's 3rd-largest PC vendor
- **Employees**: ~120,000 globally
- **Key subsidiaries**: Dell (client), Dell EMC (infrastructure), VMware (spun off 2021), Secureworks, Boomi (sold 2021)

## Business Segments

### Infrastructure Solutions Group (ISG)
- **PowerEdge servers**: Rack, tower, modular (MX series). Intel Xeon / AMD EPYC. AI-optimized: PowerEdge XE9680 (8x NVIDIA GPUs)
- **PowerStore / PowerScale / PowerFlex**: Unified, scale-out, software-defined storage
- **PowerProtect**: Backup & data protection (DD series appliances, APEX Backup Services)
- **Networking**: PowerSwitch (25/100/400GbE), SmartFabric OS10
- **HCI**: VxRail (with VMware vSAN), Dell APEX Private Cloud

### Client Solutions Group (CSG)
- **Latitude**: Business laptops (3000/5000/7000/9000 series). Rugged: Latitude Rugged Extreme
- **OptiPlex**: Business desktops (Micro, SFF, Tower, AIO)
- **Precision**: Workstations (mobile + tower). ISV-certified for CAD, AI/ML, VFX
- **XPS**: Premium consumer/prosumer laptops
- **Inspiron**: Consumer laptops & desktops
- **Vostro**: (discontinued 2023, merged into Inspiron/Latitude)
- **Monitors**: UltraSharp (USB-C hubs), P-series, S-series, gaming (Alienware)

### Dell Technologies Services
- **ProSupport / ProSupport Plus**: 24/7, predictive issue detection, SupportAssist telemetry
- **ProDeploy / ProDeploy Plus**: Factory config, imaging, BIOS settings, asset tagging, on-site deployment
- **Managed Services**: Dell APEX (as-a-service for compute, storage, networking, backup, desktops)
- **Financial Services (DFS)**: Leasing, payment plans, APEX flex-on-demand consumption, trade-in/recycling (Dell Reconnect)
- **Dell Expert Network**: SMB advisory + dedicated account reps for small businesses

## APIs & Integration Points
- **iDRAC (Integrated Dell Remote Access Controller)**: REST API for server management (Redfish standard)
- **OpenManage Enterprise**: REST API for fleet management, alerts, firmware updates
- **Dell APEX Console API**: Programmatic provisioning of as-a-service infrastructure
- **SupportAssist API**: Telemetry + case management integration
- **Dell Premier API**: Custom catalog, quoting, order tracking
- **Dell TechDirect API**: Dispatch, parts, warranty lookup
- **BIOS/Firmware**: Dell Command | Update, Dell Command | Configure (scriptable)
